Среда .NET Framework состоит из следующих основных компонентов:

1. Common Language Runtime (CLR) - исполняющая среда, которая управляет выполнением программ на языках .NET.
2. Framework Class Library (FCL) - библиотека классов, предоставляющая обширный набор функциональности для разработки приложений.
3. ASP.NET - технология для создания веб-приложений на платформе .NET.
4. Windows Presentation Foundation (WPF) - фреймворк для создания графических пользовательских интерфейсов.
5. Windows Communication Foundation (WCF) - фреймворк для создания распределенных и служб ориентированных на сообщения.
6. Windows Workflow Foundation (WF) - фреймворк для создания рабочих процессов.
7. LINQ (Language Integrated Query) - интегрированный языковой запрос, который позволяет использовать 
   SQL-подобные запросы непосредственно в коде на C# или VB.NET.
