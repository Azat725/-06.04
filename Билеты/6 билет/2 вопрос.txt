Назначение служб SQL Server и создание группы сервера:

1. MSSQLServer (SQL Server Database Engine):
   - Основное назначение: Обеспечивает хранение, обработку и обеспечение доступа к данным в базе данных SQL Server.
   
2. SQLServerAgent:
   - Основное назначение: Служба планирования и автоматизации задач SQL Server, таких как резервное копирование баз данных, 
   выполнение процедур и агентских заданий.
   
3. Microsoft Search:
   - Основное назначение: Поисковая служба SQL Server, предназначенная для обеспечения полнотекстового поиска данных в базах данных SQL Server.
   
4. Microsoft Distributed Transaction Coordinator (MSDTC):
   - Основное назначение: Координирует распределенные транзакции между несколькими источниками данных, обеспечивая целостность транзакций.

Создание группы сервера:
    Для создания группы сервера в SQL Server можно использовать функционал "Политик безопасности". Ниже приведены общие шаги по 
    созданию группы сервера:

1. Откройте Microsoft SQL Server Management Studio и подключитесь к серверу баз данных.
2. Перейдите в раздел "Политика безопасности", который находится в дереве объектов сервера.
3. Нажмите правой кнопкой мыши на "Группы сервера" и выберите "Создать новую группу сервера".
4. Задайте имя для новой группы сервера и добавьте необходимых пользователей или логины в эту группу.
5. Выберите необходимые разрешения и настройки безопасности для созданной группы.
6. Сохраните изменения.
