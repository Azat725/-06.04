- Master: База данных Master содержит метаданные и конфигурационную информацию для управления экземпляром SQL Server. 
В ней хранятся системные базы данных, системные объекты и информация о базах данных, которые находятся под управлением данного экземпляра сервера.

- Model: База данных Model используется в качестве шаблона для создания новых баз данных. Когда создается новая база данных, 
она основывается на структуре и параметрах базы данных Model.

- Tempdb: Tempdb является временной базой данных, используемой для хранения временных объектов, временных таблиц, промежуточных 
результатов запросов и другой временной информации, которая может потребоваться при выполнении операций в SQL Server.

- Msdb: База данных Msdb используется для хранения информации о заданиях планировщика заданий SQL Server, резервных копиях, 
обслуживании баз данных, истории выполнения заданий и оповещениях. В ней также хранится информация о службах интеграции, 
службах агента SQL Server и других компонентах SQL Server.
